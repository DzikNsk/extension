Thanks for downloading the Mooltipass Extension

For this extension to work correctly, you must disable all other password management applications. That way, nothing will interfere with the Mooltipass Extension.

To disable other password management extensions on Opera follow these instructions.

Part 1: Disabling 3rd Party Password Managers
Step 1: Click the opera logo in the top left corner of the browser. On the drop down menu that appears, choose "Extensions". Alternatively, press Ctrl+Shift+E to enter the extensions menu. After you have entered the extension menu, select "All" to show all installed extensions.

Here you should be able to see all of the Opera extensions you have installed. If not, try repeating Step 1 again and see if you yield different results.

Step 2: Find any other password managing applications, such as Dashlane or Lastpass. You need to remove these extensions from your computer. These extensions are displayed in boxes such as the one below:



Step 3: Finally, remove the extension by pressing the "X" at the top right corner of the screen. Confirm that you really want to remove the extension.
Be sure to repeat these steps for all other password extensions.

Part 2: Disabling Opera's Password Manager
You might have noticed that the Opera browser asks you if you would like to store credentials for a website, even if you have not installed any management extensions. These credentials are autofilled into corresponding login fields on a website, which can cause the Mooltipass extension to perform undesirably.
Step 1: In the URL bar, copy and paste in this: "opera://settings/passwords".

Step 2: Uncheck the switch for "Offer to save passwords" and "Auto Sign-in".Congratulations, you have successfully set up the Mooltipass Extension!
