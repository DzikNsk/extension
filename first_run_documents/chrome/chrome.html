Thanks for downloading the Mooltipass Extension

For this extension to work correctly, you must disable all other password management applications. That way, nothing will interfere with the Mooltipass Extension.

To disable other password management extensions on Google Chrome follow these instructions.

Part 1: Disabling 3rd Party Password Managers
Step 1: Click the menu icon ( ) on the top right corner of the browser. On the drop down menu that appears, choose "More tools", then choose "Extensions". This will bring you to Google Chrome's extension management page.

Here you should be able to see all of the Chrome Extensions you have installed. If not, try repeating Step 1 again and see if you yield different results.

Step 2: Find any other password managing applications, such as Dashlane or Lastpass. You need to remove these extensions from your computer. These extensions are displayed in rectangles such as the one below:



Step 3: Finally, remove the extension by pressing the remove button. Google Chrome will ask you one more time if you are sure, then the extension will be removed.


Be sure to repeat these steps for all other password extensions.

Part 2: Disabling Chrome's Password Manager
You might have noticed that the Google Chrome browser asks you if you would like to store credentials, even if you have not installed any management extensions. These credentials are autofilled into corresponding login fields on a website, which can cause the Mooltipass extension to perform undesirably.
Step 1: Click the menu icon ( ) on the top right corner of the browser. On the drop down menu that appears, choose "Settings".

Step 2: In the settings page of Google Chrome, click the passwords button. It looks like this:


Step 3: Uncheck both sliders for "Offer to save passwords" and "Auto Sign-in". Congratulations, you have successfully set up the Mooltipass Extension!
